# Cartier – Explainable Web Attack Detection (WAF)

Cartier is an AI-powered Web Application Firewall that detects
and explains web attacks such as:

• SQL Injection
• Cross-Site Scripting (XSS)
• Server-Side Request Forgery (SSRF)
• Remote Code Execution (RCE)

Features:

✔ Hybrid detection engine (Rules + ML)
✔ Explainable AI evidence
✔ Real-time dashboard
✔ Attack analytics
✔ REST API

Tech Stack

Backend:
FastAPI
Python
Scikit-learn

Frontend:
React.js

Database:
MongoDB / PostgreSQL

Usage

Run backend:

uvicorn backend.main:app --reload

Run frontend:

npm start
